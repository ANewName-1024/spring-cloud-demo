   - 简化后的 pom.xml 不包含 validation 和 security 依赖
   - 原有代码文件（ConfigSecurityConfig.java, SysConfig.java, ConfigController.java, ConfigJwtAuthenticationFilter.java）依赖这些库

   - 简化 ConfigServiceApplication.java 为最小可运行版本
   - 用户需要手动在终端执行:
